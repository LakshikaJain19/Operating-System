Operating Systems Lab (CSE)
This repository contains the implementation of core Operating System concepts as part of the academic curriculum.

🚀 Experiments List
Linux Basics - Navigation and file commands.
Process Mgmt - ps, kill, and top commands.
Forking - Parent and Child process creation.
FCFS - First Come First Served Scheduling.
SJF/SRTF - Shortest Job First (Preemptive & Non-Preemptive).
Priority - Priority Scheduling (Preemptive & Non-Preemptive).
Banker's Algorithm - Deadlock avoidance strategy.
FIFO - First-In-First-Out Page Replacement.
Shell Intro - Variables and arithmetic in Bash.
Shell Logic - If-Else conditionals in scripts.
🛠️ How to Run
For C programs: gcc filename.c -o output && ./output
For Shell scripts: chmod +x filename.sh && ./filename.sh
